MRSS Feed URL Generator

Generate Media RSS feed URLs for YouTube channels and playlists

YouTube Direct

YouTube provides official RSS/Atom feeds. Paste a channel ID (starts with UC) or playlist ID.

Note: The official feed only includes the 15 most recent videos. For more videos, use the API-backed feed below.

Uses the YouTube Data API via this app's Node server. Run npm start and set YOUTUBE_API_KEY in youtube.env.

YouTube feeds use https://www.youtube.com/feeds/videos.xml. API-backed feed uses YouTube Data API v3.
